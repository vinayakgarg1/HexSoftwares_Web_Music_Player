# 🎵 HexSoftwares Web Music Player

A modern, responsive, and aesthetically pleasing web-based music player. This project features a **Glassmorphic UI**, dynamic animated backgrounds, and a functional playlist system.

## ✨ Features
* **Glassmorphic Design:** A sleek, "frosted glass" interface for a premium feel.
* **Dynamic Backgrounds:** Smoothly animated gradient backgrounds that enhance the user experience.
* **Playlist Management:** Easily navigate through your favorite tracks.
* **Full Playback Controls:** Play, pause, skip, and volume adjustment.
* **Album Artwork:** Visual representation for every song in your library.

## 🛠️ Technologies Used
* **HTML5:** Structural foundation and Audio API.
* **CSS3:** Advanced styling including Flexbox, Animations, and Backdrop-filters.
* **JavaScript (Vanilla):** Logic for playback handling, DOM manipulation, and playlist control.
